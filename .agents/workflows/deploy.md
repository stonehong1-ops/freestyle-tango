// turbo-all
1. 프로젝트 빌드 확인 (npm run build)
```powershell
npm run build
```

2. Vercel 운영 서버 배포 (npx vercel --prod --yes)
```powershell
npx -y vercel --prod --yes
```

3. 배포 결과 확인 및 사용자 알림
   - Aliased URL ([freestyle-tango.kr](https://freestyle-tango.kr)) 확인 후 작업 완료 보고
